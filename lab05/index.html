Lab 02 Reflection

In this lab, I learned how HTML gives structure to a web page. I practiced using headings, paragraphs, and lists to build my first blog layout. This helped me understand how tags work together to create readable content.

Lab 03 Reflection

During Lab 03, I worked on CSS styling and saw how colors, fonts, and spacing can change the mood of a page. It was fun to see my plain HTML page come to life with simple CSS rules. I also learned how linking an external style sheet keeps my code cleaner.

Lab 05 Reflection - Best Design Practices

For Lab 05, I focused on making my page responsive. Using Flexbox and a media query, my layout now changes from one column on mobile to multiple columns on larger screens. I learned that good design should look simple and clear on every device size.
