Welcome

This website is my final project for CSIT 121. The goal of this project was to use everything I learned during the semester and put it all together into one website. I used multiple HTML pages, a shared CSS file, and navigation links to connect all the pages.

Throughout the course, I learned how important it is to keep a website organized. I focused on using headings, sections, and lists so the information is easy to read and understand. Each page follows the same layout so the site feels consistent. I also made sure the website works on different screen sizes. When the screen gets smaller, the layout changes so the content is still readable on mobile devices.
